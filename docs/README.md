## 🚀 **Inicio Rápido**

```bash
# Clonar el repositorio
git clone https://github.com/DanSRM02/Bite-of-Humor.git

# Instalar dependencias
npm install

# Configurar variables de entorno
cp .env.example .env.local

# Iniciar desarrollo
npm run dev
```
